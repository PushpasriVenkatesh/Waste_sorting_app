## About the Project

This project was developed as part of a hackathon to address real-world waste segregation problems.
AI tools (Google AI Studio) were used for prototyping and guidance, while the application logic,
UI customization, and feature integration were implemented and modified by the developer.

# Run and deploy your AI Studio app

This contains everything you need to run your app locally.

View your app in AI Studio: https://ai.studio/apps/drive/1yYgYijKlmHjwtBB7OJQkyRVosXAoLgiQ

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
